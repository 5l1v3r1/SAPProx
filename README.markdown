#4. Description
SAPProx is a proof of concept tool for intercepting and modifying SAP GUI (DIAG protocol) traffic.

#5. Requirements
Java runtime environment.  
Custom JNI Library.
Binary builds of the JNI library are only available for the  

following platforms:  
Mac OS/X  
Windows (32-bit)  
Linux (32-bit)
